I see that currently recurring tasks are not intuitive. When you define a recurring action, actually you’re defining a recurring pattern from which normal tasks are created. So when you edit a recurring task, you’re not editing the already created tasks that are in tickler/context/home, but you’re editing the pattern. This is why the change is not reflected anywhere else.

In this case you need to edit both :-(

I guess this needs some improvements…
