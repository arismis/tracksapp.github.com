I have played around with this and found some additional information.

Setting :source for config.gem makes rake gems:install find the actionwebservice. It can then install it.

config.gem 'datanoise-actionwebservice', :lib => 'actionwebservice', :source => "http://gems.github.com" 

Falling back on a default actionwebservice (problem with datanoise-actionwebservice is that is abandoned) as follows:

config.gem 'actionwebservice' 

Makes it install too.
