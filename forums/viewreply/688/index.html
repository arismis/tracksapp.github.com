I fixed this by changing the views/todos/tag.html.erb

Replaced this

<%= render :partial => "contexts/context", :collection => @contexts_to_show,        
            :locals => { :collapsible => true } %> 

with this

<%= render :partial => "contexts/context", :collection => @contexts,        
            :locals => { :collapsible => true } %> 

on line 9
