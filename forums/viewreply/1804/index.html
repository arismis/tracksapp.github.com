I have found that the newer version 1.7 uses a random password for MySQL.

Go to /opt/tracks-1.7-1/apps/tracks/config and look in the contents of the file database.yml it will have the password embeded: 

production:
  adapter: mysql
  database: bitnami_tracks
  host: localhost
  username: bitnami
  password: xxxxxxxxxx
  socket: /opt/tracks-1.7-1/mysql/tmp/mysql.sock


use that name and password if you need to log into the MySql base.

Also don’t forget if you are doing two installations more than likely if you run the command mysql it will use the Ubuntu package installed version at /usr/bin.  You need to go to the directory /opt/tracks-1.7-1/mysql/bin an run with ./mysql
